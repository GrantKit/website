# GrantKit Website

Marketing site for GrantKit at [grantkit.io](https://grantkit.io).

## Tech stack

- Astro
- TailwindCSS
- Deployed via Vercel

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build
```

## Related repos

- [app](https://github.com/GrantKit/app) - Web application at app.grantkit.io
- [grantkit](https://github.com/GrantKit/grantkit) - Python CLI for local sync
